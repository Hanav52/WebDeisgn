# 동화약품 리뉴얼 프로젝트

이 프로젝트는 오래된 디자인을 사용하는 동화약품의 사이트의 리뉴얼을 진행했습니다. 
기획부터 시작하여 디자인 -> 구현 순서대로 진행하였습니다.

[동화약품 리뉴얼](https://hanav52.github.io/WebDeisgn/public/mainindex.html).

>

>
> Figma, Photoshop 등을 사용하여 디자인 초안 구현
>
> Git을 이용하여 배포
